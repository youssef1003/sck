## ⚠️ ما يحتاج تحسين (اختياري - ليس ضروري للـ Production)

### 1. الـ APIs الناقصة (اختياري)

#### أ) API لإدارة الحجوزات من الـ Admin
**الحالة:** الـ Frontend موجود، لكن الـ API ناقص

**المطلوب:**
- `GET /api/admin/bookings` - جلب جميع الحجوزات
- `PATCH /api/admin/bookings/:id` - تحديث حالة الحجز (pending → confirmed)
- `DELETE /api/admin/bookings/:id` - حذف حجز

**الأولوية:** متوسطة (يمكن إضافتها لاحقاً)

---

#### ب) API لإدارة الرسائل من الـ Admin
**الحالة:** الـ Frontend موجود، لكن الـ API ناقص

**المطلوب:**
- `GET /api/admin/contacts` - جلب جميع الرسائل
- `PATCH /api/admin/contacts/:id` - تحديث حالة الرسالة (new → resolved)
- `DELETE /api/admin/contacts/:id` - حذف رسالة

**الأولوية:** متوسطة (يمكن إضافتها لاحقاً)

---

#### ج) API لإدارة المدونة
**الحالة:** الـ Frontend موجود، لكن الـ API ناقص

**المطلوب:**
- `GET /api/admin/blog` - جلب جميع المقالات
- `POST /api/admin/blog` - إنشاء مقالة جديدة
- `PATCH /api/admin/blog/:id` - تعديل مقالة
- `DELETE /api/admin/blog/:id` - حذف مقالة

**الأولوية:** منخفضة (المدونة ليست أساسية)

---

#### د) API لإدارة التوظيف
**الحالة:** الـ Frontend موجود، لكن الـ API ناقص

**المطلوب:**
- `GET /api/admin/careers` - جلب جميع طلبات التوظيف
- `PATCH /api/admin/careers/:id` - تحديث حالة الطلب
- `DELETE /api/admin/careers/:id` - حذف طلب

**الأولوية:** متوسطة (إذا كنت تستخدم نظام التوظيف)

---

#### هـ) API لحفظ محتوى الصفحات المدارة

**المطلوب:**

- `POST /api/admin/content/:page` - حفظ محتوى صفحة

**الصفحات:**
- Home (الصفحة الرئيسية)
- Services (الخدمات)
- About (من نحن)
- Contact (تواصل معنا)

**الأولوية:** عالية (لو عايز المحتوى يتحفظ في قاعدة البيانات)
