## 🔍 التفاصيل التقنية:

### الـ API الجديد:

**Endpoint:**
```
GET /api/blog
```

**Query Parameters:**
- `limit` - عدد المقالات (default: 100)
- `offset` - البداية (default: 0)
- `category` - التصنيف (optional)
- `search` - البحث (optional)

**Response:**
```json
{
  "success": true,
  "data": [
    {
      "id": "uuid",
      "title": "عنوان المقال",
      "excerpt": "المقتطف",
      "content": "المحتوى",
      "author": "الكاتب",
      "category": "التصنيف",
      "image_url": "رابط الصورة",
      "booking_link": "/consultation",
      "is_published": true,
      "created_at": "2026-05-01T..."
    }
  ],
  "count": 6
}
```
