# 📋 اللينكات المتاحة للمقالات

## 🎯 الاستخدام:

عند إضافة مقال جديد، اختار الرابط المناسب من القائمة أدناه وضعه في حقل **"رابط حجز الاستشارة"**.

## 🎨 أمثلة على الاستخدام:

### مثال 1: مقال عن التوظيف للباحثين عن عمل
```
العنوان: "كيف تكتب سيرة ذاتية احترافية"
رابط الحجز: /careers
نص الزر: "قدم طلبك الآن"
```

### مثال 2: مقال لأصحاب الأعمال
```
العنوان: "كيف تختار الموظف المناسب"
رابط الحجز: /employer-request
نص الزر: "اطلب موظفين"
```

### مثال 3: مقال عن الاستشارات
```
العنوان: "أهمية الاستشارات الإدارية"
رابط الحجز: /consultation
نص الزر: "احجز استشارة"
```

### مثال 4: مقال عن الخدمات
```
العنوان: "خدماتنا في الموارد البشرية"
رابط الحجز: /services
نص الزر: "اعرف المزيد"
```

## 💡 نصائح:

### 1. اختيار الرابط المناسب:
- **للمقالات العامة:** استخدم `/consultation`
- **للتوظيف (باحثين عن عمل):** استخدم `/careers`
- **للتوظيف (أصحاب أعمال):** استخدم `/employer-request`
- **للخدمات:** استخدم `/services`

### 2. نص الزر:
الزر هيظهر دايماً بنص **"احجز استشارة"** 📅

لكن ممكن في المستقبل نخليه dynamic حسب الرابط.

### 3. روابط خارجية:
لو عايز رابط خارجي، استخدم الرابط الكامل:
```
https://example.com
```

## 🎯 الخلاصة:

**عند إضافة مقال:**
1. اختار الرابط المناسب من القائمة أعلاه
2. انسخه والصقه في حقل "رابط حجز الاستشارة"
3. احفظ المقال
4. الزر هيظهر في المقال ويوجه للرابط المحدد

**مثال:**
```
رابط حجز الاستشارة: /careers
```

الزر هيظهر كده:
```
📅 احجز استشارة
```

ولما المستخدم يضغط عليه، هيروح على صفحة `/careers`
