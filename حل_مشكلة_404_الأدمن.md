# 🔧 حل مشكلة 404 في صفحات الأدمن

## ✅ التشخيص
المشكلة: عند محاولة الدخول على `/admin/dashboard` يظهر خطأ 404

**السبب**: الموقع على Vercel محتاج إعادة نشر (Redeploy) لتحديث الـ Routes

## 🎯 الحل السريع

### الطريقة 1: إعادة النشر من Vercel Dashboard (الأسرع)

1. **اذهب إلى Vercel Dashboard:**
   - https://vercel.com/dashboard

2. **اختر المشروع:**
   - ابحث عن مشروع `sck` أو `scq-platform`

3. **اذهب إلى Deployments:**
   - اضغط على تبويب "Deployments"

4. **أعد نشر آخر deployment:**
   - اضغط على النقاط الثلاث (...) بجانب آخر deployment
   - اختر "Redeploy"
   - اضغط "Redeploy" مرة أخرى للتأكيد

5. **انتظر حتى ينتهي:**
   - سيستغرق 1-2 دقيقة
   - ستظهر علامة ✅ عند الانتهاء

### الطريقة 2: Push تغيير بسيط إلى GitHub

إذا كنت تستخدم Git:

\`\`\`bash
# في terminal
git add .
git commit -m "Fix: Trigger redeploy for admin routes"
git push origin main
\`\`\`

Vercel سيعمل auto-deploy تلقائياً.
