# نظام المصادقة والتسجيل | Authentication System

## ✅ تم الإنجاز | Completed

### 1. صفحات المصادقة | Authentication Pages

#### صفحة التسجيل | Register Page (`/register`)
- نموذج تسجيل كامل مع التحقق من البيانات
- حقول: الاسم الكامل، البريد الإلكتروني، الهاتف، كلمة المرور، تأكيد كلمة المرور
- التحقق من تطابق كلمات المرور
- التحقق من طول كلمة المرور (6 أحرف على الأقل)
- التحقق من عدم تكرار البريد الإلكتروني
- تسجيل دخول تلقائي بعد التسجيل
- دعم كامل للغة العربية والإنجليزية

#### صفحة تسجيل الدخول | Login Page (`/login`)
- نموذج تسجيل دخول بسيط
- حقول: البريد الإلكتروني، كلمة المرور
- إظهار/إخفاء كلمة المرور
- رابط للتسجيل
- رابط لتسجيل دخول الإدارة
- رابط للعودة للصفحة الرئيسية

### 4. نظام التخزين | Storage System

#### localStorage Keys
```javascript

// بيانات المستخدم الحالي

'scq_user_data' // Current user data (JSON)

// طلبات التوظيف

```
