# إصلاح مشاكل لوحة التحكم - دليل شامل

## المشكلة الحالية
لوحة التحكم للإدارة تظهر أخطاء 500 لأن بعض الجداول المطلوبة غير موجودة في قاعدة البيانات.

## الحل السريع

### الخطوة 1: تشغيل Migration Script في Supabase

1. اذهب إلى [Supabase Dashboard](https://supabase.com/dashboard)
2. اختر مشروعك
3. اذهب إلى **SQL Editor**
4. انسخ والصق المحتوى من ملف `MISSING_TABLES_MIGRATION.sql`
5. اضغط **Run** لتنفيذ الكود

### الخطوة 2: التحقق من النتائج

بعد تشغيل الـ Migration، ستحصل على:
- ✅ جدول `contact_requests` مع 3 رسائل تجريبية
- ✅ جدول `consultation_bookings` مع 3 حجوزات تجريبية  
- ✅ جدول `blog_posts` مع 3 مقالات تجريبية
- ✅ جدول `services` مع 4 خدمات تجريبية
- ✅ جداول `ai_conversations` و `ai_messages` للدردشة الذكية

### الخطوة 3: اختبار لوحة التحكم

1. اذهب إلى موقعك: https://sck-tawny.vercel.app
2. سجل دخول بالبيانات:
   - **Email**: admin@sck.com
   - **Password**: scq2025
3. تحقق من أن الإحصائيات تظهر بشكل صحيح

## ما تم إصلاحه في الكود

### 1. تحسين Admin API
- إضافة معالجة أخطاء أفضل في `/api/admin/stats`
