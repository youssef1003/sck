## ⚠️ الأخطاء المحتملة وحلولها

### خطأ 1: "Database not configured"
**السبب:** Environment Variables مش موجودة في Vercel

**الحل:**
1. افتح Vercel Dashboard
2. اختر مشروع SCK
3. اذهب إلى Settings → Environment Variables
4. تأكد من وجود:
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_KEY`
   - `JWT_SECRET`

---

### خطأ 2: "Table page_content does not exist"
**السبب:** لم يتم تشغيل SQL Script في Supabase

**الحل:**
- ارجع للخطوة 1 وشغّل SQL Script

---

### خطأ 3: "Function update_updated_at does not exist"
**السبب:** الـ function مش موجودة في قاعدة البيانات

**الحل:**
1. افتح Supabase SQL Editor
2. شغّل ملف `DATABASE_SETUP_SIMPLE.sql` الأول
3. بعدين شغّل `DATABASE_CONTENT_TABLE.sql`

---

### خطأ 4: 404 Not Found
**السبب:** Vercel لسه مخلصش النشر

**الحل:**
- انتظر دقيقتين وحاول تاني
