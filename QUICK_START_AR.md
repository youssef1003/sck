## 📍 خطوتين فقط وخلاص:

### الخطوة 1️⃣: Supabase (دقيقة واحدة)

1. افتح: https://supabase.com/dashboard
2. اختر مشروعك
3. اضغط **SQL Editor** من القائمة اليسرى
4. **انسخ والصق** هذا الكود:

```sql
ALTER TABLE blog_posts ADD COLUMN IF NOT EXISTS booking_link TEXT;
UPDATE blog_posts SET booking_link = '/consultation' WHERE booking_link IS NULL;
SELECT 'تم بنجاح! ✅' as result;
```

5. اضغط **Run** أو **F5**
6. لو ظهرت رسالة "تم بنجاح! ✅" يبقى تمام ✅
