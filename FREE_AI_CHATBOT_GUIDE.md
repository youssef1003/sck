## 📋 الخطوات للتشغيل

### الخطوة 1: تشغيل SQL Script (5 دقائق)

**في Supabase SQL Editor:**
1. افتح ملف `RAG_DATABASE_SIMPLE.sql`
2. انسخ كل المحتوى
3. الصقه في SQL Editor
4. اضغط Run

**ده هيعمل:**
- ✅ إنشاء 3 جداول
- ✅ إضافة محتوى تجريبي
- ✅ تفعيل البحث النصي

---

### الخطوة 2: إضافة Hugging Face API Key (اختياري - 5 دقائق)

**للحصول على أداء أفضل:**

1. اذهب إلى: https://huggingface.co/settings/tokens
2. سجل دخول (مجاني)
3. اضغط "New token"
4. انسخ الـ token

**في Vercel:**
1. Settings → Environment Variables
2. أضف:
   ```
   HF_API_KEY = hf_...
   ```
3. Save

**ملاحظة:** 
- ✅ **اختياري** - الشات يعمل بدونه
- ✅ **مجاني** - بدون تكلفة
- ✅ **أفضل** - ردود أسرع وأذكى

---

### الخطوة 3: Redeploy في Vercel (دقيقة واحدة)

1. افتح Vercel Dashboard
2. اختر مشروع SCK
3. Deployments → اضغط "..." → Redeploy

**انتظر 2-3 دقائق**

---

### الخطوة 4: اختبار (دقيقة واحدة)

1. افتح الموقع: https://sck-tawny.vercel.app
2. شوف زر الشات (أسفل يمين)
3. اضغط عليه
4. اكتب سؤال: "ما هي خدماتكم؟"
5. انتظر الرد

**المتوقع:**
- ✅ رد ذكي عن الخدمات
- ✅ بالعربي أو الإنجليزي
- ✅ بدون أخطاء
