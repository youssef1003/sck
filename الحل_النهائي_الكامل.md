## 🚀 الخطوات اللي عليك دلوقتي

### 1️⃣ استنى النشر يخلص في Vercel
- افتح: https://vercel.com/dashboard
- شوف حالة النشر
- استنى لحد ما يطلع "Ready"

### 2️⃣ **مهم جداً**: شغل SQL في Supabase

⚠️ **لازم تعمل ده قبل ما تجرب الموقع**:

1. افتح Supabase Dashboard
2. روح SQL Editor
3. شغل الملفات دي بالترتيب:

**أولاً**: افتح ملف `DATABASE_FIX_SIMPLE.sql` وشغله
**ثانياً**: افتح ملف `CREATE_LOGIN_FUNCTION.sql` وشغله

4. تأكد إن اشتغلوا صح:
```sql
-- شوف لو الـ admin موجود
SELECT id, email, role, is_active FROM users WHERE email = 'admin@sck.com';

-- جرب الـ login function
SELECT * FROM login_user('admin@sck.com', 'Admin@123456');
```

لازم يرجع بيانات المستخدم (id, email, full_name, إلخ)

### 3️⃣ تأكد من المتغيرات في Vercel

روح: Vercel Dashboard > Settings > Environment Variables

تأكد إن موجود:
```
✓ SUPABASE_URL
✓ SUPABASE_SERVICE_KEY
✓ JWT_SECRET
```

اختياري (للـ AI):
```
○ HF_API_KEY (أو OPENAI_API_KEY)
```

### 4️⃣ جرب الموقع

#### A) جرب الـ API من Terminal:

```bash
# 1. Health Check
curl https://sck-tawny.vercel.app/api/health

# 2. Login
curl -X POST https://sck-tawny.vercel.app/api/auth?action=login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@sck.com","password":"Admin@123456"}'

# 3. AI Chat
curl -X POST https://sck-tawny.vercel.app/api/ai/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"مرحبا","language":"ar"}'
```

#### B) جرب في المتصفح:

1. **افتح الموقع**: https://sck-tawny.vercel.app
2. **افتح DevTools** (F12)
3. **روح Console**
4. **جرب AI Assistant**:
   - اضغط على الزرار أسفل يمين
   - اكتب رسالة وابعت
   - شوف Console - مفيش أخطاء
   - AI بيرد صح
5. **جرب Login**:
   - روح: https://sck-tawny.vercel.app/login
   - سجل دخول: admin@sck.com / Admin@123456
   - تسجيل الدخول نجح
   - اتوجهت للـ Dashboard
