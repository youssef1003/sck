## 🎯 كيف تستخدم الميزة:

### الطريقة:
1. افتح **Dashboard** → **إدارة المدونة**
2. اضغط على **تعديل** للمقال (أو **مقال جديد**)
3. في حقل **"رابط حجز الاستشارة"**، ضع الرابط المناسب
4. في حقل **"نص الزر"**، اكتب النص اللي عايزه يظهر على الزر
5. اضغط **"تحديث المقال"** أو **"نشر المقال"**

### أمثلة:

#### مثال 1: مقال للموظفين
```
العنوان: "10 نصائح لكتابة CV احترافي"
رابط الحجز: /careers
نص الزر: قدم طلبك الآن
```
**النتيجة:** الزر يقول "قدم طلبك الآن" ويوجه لصفحة الوظائف

#### مثال 2: مقال لأصحاب الأعمال
```
العنوان: "كيف تبني فريق عمل ناجح"
رابط الحجز: /employer-request
نص الزر: اطلب موظفين
```
**النتيجة:** الزر يقول "اطلب موظفين" ويوجه لصفحة طلب موظفين

#### مثال 3: مقال عن الاستشارات
```
العنوان: "أهمية الاستشارات الإدارية"
رابط الحجز: /consultation
نص الزر: احجز استشارة
```
**النتيجة:** الزر يقول "احجز استشارة" ويوجه لصفحة الاستشارات

#### مثال 4: نص مخصص
```
العنوان: "دورة تدريبية في الإدارة"
رابط الحجز: /contact
نص الزر: سجل في الدورة
```
**النتيجة:** الزر يقول "سجل في الدورة" ويوجه لصفحة التواصل

## 🐛 حل المشاكل:

### المشكلة: حقل "نص الزر" مش ظاهر في Dashboard
**الحل:**
1. تأكد من عمل Deploy للتحديثات
2. امسح Cache المتصفح (Ctrl+Shift+Delete)
3. جرب في نافذة Incognito
4. تأكد من تحديث الصفحة (F5)

### المشكلة: الزر لسه بيقول "احجز استشارة"
**الحل:**
1. تأكد من تشغيل SQL في Supabase
2. تأكد من حفظ التعديلات في Dashboard
3. تحقق من قاعدة البيانات:
```sql
SELECT button_text FROM blog_posts WHERE id = 'المقال_ID';
```
4. امسح Cache المتصفح

### المشكلة: خطأ عند حفظ المقال
**الحل:**
1. تأكد من تشغيل SQL في Supabase أولاً
2. تحقق من Console (F12) لأي أخطاء
3. تأكد من أن عمود `button_text` موجود:
```sql
SELECT column_name 
FROM information_schema.columns 
WHERE table_name = 'blog_posts' 
  AND column_name = 'button_text';
```
