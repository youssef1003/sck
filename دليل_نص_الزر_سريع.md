# 📋 دليل سريع: تخصيص نص الزر

## 🎯 الميزة الجديدة:

---

## 📝 الخطوات:

### 1. شغل SQL في Supabase (مرة واحدة فقط):
```
1. افتح Supabase
2. اضغط على SQL Editor
3. افتح ملف ADD_BUTTON_TEXT_COLUMN.sql
4. انسخ كل المحتوى
5. الصقه في SQL Editor
6. اضغط Run
```

### 2. Deploy التحديثات:
```bash
git add .
git commit -m "Add custom button text"
git push
```

### 3. استخدم الميزة:
```
1. افتح Dashboard → إدارة المدونة
2. اضغط تعديل على المقال
3. في حقل "نص الزر"، اكتب النص اللي تحبه
4. احفظ التعديلات
```
