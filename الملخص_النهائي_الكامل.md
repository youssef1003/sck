## 🐛 حل المشاكل الشائعة:

### المشكلة 1: حقل "نص الزر" مش ظاهر في Dashboard
**الأسباب المحتملة:**
- لم يتم Deploy التحديثات
- Cache المتصفح قديم

**الحل:**
```
1. تأكد من عمل git push
2. تأكد من نجاح Deployment في Vercel
3. امسح Cache المتصفح (Ctrl+Shift+Delete)
4. جرب في نافذة Incognito
5. حدث الصفحة (F5)
```

### المشكلة 2: الزر لسه بيقول "احجز استشارة"
**الأسباب المحتملة:**
- لم يتم تشغيل SQL في Supabase
- لم يتم حفظ التعديلات

**الحل:**
```
1. تأكد من تشغيل SQL في Supabase
2. تحقق من قاعدة البيانات:
   SELECT button_text FROM blog_posts WHERE id = 'المقال_ID';
3. تأكد من حفظ التعديلات في Dashboard
4. امسح Cache المتصفح
5. جرب في نافذة Incognito
```

### المشكلة 3: خطأ 404 عند الضغط على الزر
**الأسباب المحتملة:**
- الرابط غلط
- الصفحة مش موجودة
- لم يتم Deploy

**الحل:**
```
1. تأكد من كتابة الرابط صح (بالـ / في الأول)
2. استخدم الروابط من الجدول أعلاه
3. تأكد من نجاح Deployment
4. امسح Cache المتصفح
```

### المشكلة 4: خطأ عند حفظ المقال
**الأسباب المحتملة:**
- عمود `button_text` مش موجود في قاعدة البيانات

**الحل:**
```
1. تأكد من تشغيل SQL في Supabase أولاً
2. تحقق من وجود العمود:
   SELECT column_name 
   FROM information_schema.columns 
   WHERE table_name = 'blog_posts' 
     AND column_name = 'button_text';
3. لو مش موجود، شغل SQL تاني
4. تحقق من Console (F12) لأي أخطاء
```
