## 🚀 كيفية العرض (5 دقائق)

### الخطوة 1: تشغيل المشروع (30 ثانية)
```bash
# Terminal 1 - Frontend
cd frontend
npm run dev

# Terminal 2 - Backend
cd backend
uvicorn main:app --reload
```

### الخطوة 2: عرض الصفحة الرئيسية (1 دقيقة)
1. افتح `http://localhost:3000`
2. أظهر:
   - Hero Section
   - الخدمات
   - الإحصائيات
   - آراء العملاء
   - تبديل اللغة (AR/EN)

### الخطوة 3: عرض User Dashboard (1 دقيقة)
1. سجل حساب جديد كـ Client
2. احجز استشارة من `/services`
3. أرسل رسالة من `/contact`
4. ارجع لـ `/dashboard` وأظهر:
   - الملف الشخصي
   - الحجوزات
   - الرسائل
   - تعديل البيانات

### الخطوة 4: عرض Employer Dashboard (1 دقيقة)
1. سجل حساب جديد كـ Employer
2. اذهب لـ `/employer/dashboard`
3. أظهر:
   - الإحصائيات
   - قائمة المتقدمين
   - البحث والفلترة
   - تفاصيل المتقدم
   - أزرار التواصل

### الخطوة 5: عرض Admin Panel (1.5 دقيقة)
1. سجل دخول كـ Admin (`admin` / `scq2025`)
2. اذهب لـ `/admin/home` وأظهر:
   - تعديل Hero Section
   - تعديل الإحصائيات
   - إضافة/حذف خدمة
   - إضافة/حذف تعليق
   - حفظ التغييرات
3. اذهب لـ `/admin/employers` وأظهر:
   - قائمة أصحاب العمل
   - الموافقة/الرفض
   - الإحصائيات
